Version 3.0 duplicate referrals
Summary

The process is the same for this version as it is for version 2.0 below. The only difference being that this is now aligned with the current prototype journey.

For iteration 1, we will only be able to identify duplicates at the end of the journey. Instead of a confirmation page, the user will see an error screen. The system will only identify duplicates from any referrals that have gone through our system previously. The user should still be checking LMS and F&R for duplicates across other systems.

Version 1.0 duplicate referrals prototype
Sprint version
Sprint 0 Beta
Summary
If a citizen has already been referred to a national contracted provision they are not eligible to be referred to any other national contracted provision unless they either complete the programme or, the provider rejects them or cancels the referral. The technical solution can identify the person and their referral history at the starting point of the service and the UI prevents the work coach from trying to refer the citizen unnecessarily.

Supporting information

Sprint version
Beta Sprint 24
Summary of design

We have created two versions of a 3rd party referral journey.

V1.0 submits name and NINO to CIS and, assuming we can verify them with this, returns the address and contact details for the submitted profile. This forms part of the SO SRA component available functionality. However, we believe this is open to potential exploitation as it allows users to look for anyone's details for whom they have name and NINO, thereby giving users potential to look up anyone.

V1.1 asks for all of a citizen's personal information up front. At the point of submitting the DoB we check the details match and only 'intervene' if the details don't match or if the citizen is already on provision. We then ask for address and contact details. Assuming the majority of details submitted are correct and they aren't on provision, the user will assume they are just submitting referral details and we don't have to expose any unnecessary citizen data.
