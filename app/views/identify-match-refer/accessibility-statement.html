Accessibility statement for Support Offers

This service is part of the wider GOV.UK website. There’s a separate accessibility statement for the main GOV.UK website.

This statement only applies to the Support Offers service.

Using this service

This service is run by the Department of Work and Pensions (DWP). We want as many people as possible to be able to use this service. You should be able to:

  • change colours, contrast levels and fonts using browser or device settings
  • zoom in up to 400% without the text spilling off the screen
  • use your keyboard to navigate the service
  • use the service with a screen-reader (including the recent versions of VoiceOver)

We've also made the text in the service as simple as possible to understand. AbilityNet has advice on making your device easier to use if you have a disability.

How accessible this service is

The service uses common styles, patterns and components from the GOV.UK Design System which are widely regarded as accessible.

This site was manually tested using Zoom and Voice Control.

Reporting accessibility problems with this service

If you find any problems that are not listed on this page or you think we’re not meeting the accessibility requirements, contact us on support.offers@dwp.gov.uk.

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’).

If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

Technical information about this service’s accessibility

The Department for Work and Pensions is committed to making its services accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

Compliance status

This service is fully compliant with the Web Content Accessibility Guidelines version 2.2 AA standard.

What we’re doing to improve accessibility

We will continue to iterate the design of the service, with accessibility considerations built into the lifecycle of each release.

Preparation of this accessibility statement

This statement was prepared on 12th August 2024. It was last reviewed on 14th August 2024.

This service was last tested on 9th August 2024. The testing was carried out by the Department for Work and Pensions. This service was tested using both automated and manual testing.

We tested a sample of pages that represent each type of design pattern used in this service.

Last updated 15th August 2024.
